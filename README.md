# front-personal-trainer-app
Esse projeto tem como objetivo fornecer um serviço que possa melhorar a experiência de acompanhamento dos alunos pelo Personal Trainer.

O problema que pretende resolver é a dificuldade de preenchimento de planilhas pelos alunos, fornecendo, assim, uma interface mais amigável para que os alunos possam fazer os registros de suas fichas de treino. Além disso, a intenção é que também seja mais simples que o professor faça uso da ferramenta para montar modelos de treino e fazer adaptações para as necessidades de cada aluno.

A primeira versão tem a intenção de apresentar uma estrutura bem simplificada, onde o Personal terá um perfil onde pode fazer o gerenciamento dos seus alunos, com algumas informações básicas de cadastro e que possa criar treinos para cada aluno, com um planejamento de quantas vezes a rotina se repete. Além disso o treinador poderá fazer o cadastro de exercícios para simplificar o momento de aplicá-lo a algum treino.

Os alunos terão perfis bem simples, podendo visualizar suas fichas de treino e registrar o número de repetições feitas, o peso e outras informações sobre o treino, como nível de descanso, intensidade e duração do treino.
